1. Which of the following is NOT an example of sectioning content?
    A) <section>
    B) <article>
    C) <nav>
    D) <table>

2. Which of the following is NOT an example of phrasing content?
    A) <em>
    B) <strong>
    C) <code>
    D) <section>

3. Which of the following is an example of heading content?
    A) <p>
    B) <div>
    C) <h1>
    D) <table>

Answers:

1. D) <table>
2. D) <section>
3. C) <h1>
